# ``MOQClient``

Build Media over QUIC client workflows with the `MOQClient` module.

## Overview

`MOQClient` provides client-side APIs for connecting to MOQ endpoints over QUIC and interacting with tracks, objects, and session-level behaviors built on top of `MOQCore`.

Use this module when you need to:

- Establish client-side MOQ sessions.
- Subscribe to media tracks and consume objects.
- Integrate MOQ messaging into your app-level pipeline.
- Coordinate with QUIC transport and stream lifecycle.
